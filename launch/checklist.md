# imgcrush Launch Checklist

## Pre-launch

- [ ] Clean macOS test install (fresh user account)
- [ ] Verify `brew install` works from tap
- [ ] Verify direct download + install
- [ ] Test trial flow (first run → 14 days → expiration)
- [ ] Test license activation flow
- [ ] Verify JSON output parses correctly
- [ ] Test on Intel Mac (CPU fallback)
- [ ] Landing page live at imgcrush.dev
- [ ] Gumroad product pages live (Personal, Team, Enterprise)
- [ ] Etsy product listings live (Personal, Team, Enterprise)
- [ ] Payment flow test — Gumroad (test purchase + license delivery)
- [ ] Payment flow test — Etsy (test purchase + license delivery)
- [ ] OG image shows correctly (Twitter card validator, FB debugger)
- [ ] Analytics tracking verified (Plausible)
- [ ] GitHub repo public + README polished
- [ ] v1.0.0 tag pushed → release created

## Launch Day

- [ ] Post on Hacker News (Show HN)
- [ ] Post Twitter/X thread
- [ ] Post on Reddit (r/swift, r/macOS, r/webdev)
- [ ] Submit to ProductHunt
- [ ] Post Dev.to article
- [ ] Monitor HN/Reddit comments for first 4 hours
- [ ] Reply to all comments within 1 hour
- [ ] Track sales + trial starts in real-time

## Post-launch (week 1)

- [ ] Follow up on HN/Reddit comments
- [ ] Fix any reported bugs (hotfix releases)
- [ ] Send thank-you email to first buyers
- [ ] Collect testimonials from early users
- [ ] Write "lessons learned" blog post
- [ ] Plan v1.1 based on feedback
